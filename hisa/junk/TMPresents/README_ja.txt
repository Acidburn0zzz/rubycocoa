TMPresents ReadMe
=================

これはなに？
------------

これは[高橋メソッド][1]専用プレゼンテーションツールです。
テキストファイルの内容を、できるだけ大きく表示します。


テキストの書式
--------------

* 空行がページ区切りになります。
* ページ内容全体がインデントされているとき、テキストを左揃えで表示します。

使いかた
--------

テキストファイルを開きます。

* 次のページを表示：クリックまたはてきとうなキー
* 前のページを表示：左矢印キー
* フルスクリーン表示：コマンド+F
* フルスクリーン解除：コマンド+.またはescキー

ライセンス
----------

このプログラムはkimura wataruが作成したフリーソフトウェアです。
いわゆるX11ライセンスのもとに自由に改変、複製することができます。
