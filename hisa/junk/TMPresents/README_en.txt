TMPresents ReadMe
=================

What's this?
------------

This application is a presentation tool for "Takahashi Method". 
TMPresents draws text with very large font.

Text formatting rules
--------------

* blank line: page separator
* indented text block: align left

How to use
--------

open a text file.

* show next page: click on window or any key
* show previous page: left arrow
* fullscreen: command + F
* stop fullscreen: command + . or escape key
