What is RubyCocoa ?

RubyCocoa is libraries and framework for using Cocoa Objects on MacOS X in Ruby scripts. You can use RubyCocoa for:

  • using Cocoa objects in Ruby Script.
  • writing Cocoa application by Ruby.

Suitable to use RubyCocoa:

  • A simple action check of Cocoa object
  • A simple prototyping of Cocoa Application
  • Wrapping simple OSX GUI for Ruby script

And unsuitable to use RubyCocoa:

  • Making of commercial software
  • Making of mission critical software
  • Making of long life cycle software

Script Examples

Next script is playing system sounds.

require 'osx/cocoa'
snd_files =`ls /System/Library/Sounds/*.aiff`.split
snd_files.each do |path|
  snd = OSX::NSSound.alloc.
    initWithContentsOfFile_byReference (path, true)
  snd.play
  sleep 0.25 while snd.isPlaying?
end

Next script is a class definition that is related with nib file made by Interface Builder.

require 'osx/ocobject'

class AppCtrl < OSX::OCObject

  ib_loadable
  ib_outlets :monthField, :dayField, :mulField

  def awakeFromNib
    @monthField.setIntValue  Time.now.month
    @dayField.setIntValue Time.now.day
    convert
  end

  def convert (sender = nil)
    val = @monthField.intValue * @dayField.intValue
    @mulField.setIntValue (val)
    @monthField.selectText (self)
  end

  def windowShouldClose (sender = nil)
    OSX.NSApp.stop (self)
    true
  end    

end
