What is RubyCocoa ?

RubyCocoa is a Ruby library for using Cocoa Objects on Mac OS X in Ruby scripts. You can use RubyCocoa for:

  • using Cocoa objects in Ruby Script
  • writing Cocoa application with Ruby and Interface Builder
  • exploration of Cocoa with irb
  • prototyping of a Cocoa Application
  • development of a small scale simple Cocoa Application
  • Wrapping OSX GUI for Ruby script

with Mac OS X 10.2 (Jaguar)

You can use RubyCocoa with pre-installed Ruby in Mac OS X 10.2 (Jaguar). You don't need to install extra Ruby in Jaguar.

Script Examples

Next script is playing system sounds.

require 'osx/cocoa'
snd_files =`ls /System/Library/Sounds/*.aiff`.split
snd_files.each do |path|
  snd = OSX::NSSound.alloc.
    initWithContentsOfFile_byReference (path, true)
  snd.play
  sleep 0.5
end

Next script is a class definition that is related with nib file made by Interface Builder.

require 'osx/cocoa'

class AppCtrl < OSX::NSObject

  ib_outlets :monthField, :dayField, :mulField

  def awakeFromNib
    @monthField.setIntValue  Time.now.month
    @dayField.setIntValue Time.now.day
    convert
  end

  def convert (sender = nil)
    val = @monthField.intValue * @dayField.intValue
    @mulField.setIntValue (val)
    @monthField.selectText (self)
  end

  def windowShouldClose (sender = nil)
    OSX.NSApp.stop (self)
    true
  end    

end
