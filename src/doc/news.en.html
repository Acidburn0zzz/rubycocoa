a patch fix error of config with Ruby 1.6.8 (2003-01-12)

In build of RubyCocoa 0.4 with Ruby 1.6.8, segmentation fault occur at 'ruby install.rb config'. This patch fix the problem. Apply the patch as following:

$ cd {rubycocoa-0.4.0}
$ patch -p0 < ruco0.4.0-fw-post-config.patch
